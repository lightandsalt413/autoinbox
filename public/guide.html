Paano I-connect ang Iyong Email?

Sundin ang simpleng gabay na ito para i-setup ang Gmail o Yahoo Mail gamit ang App Password.

🔑 Ano ang App Password?

Ang App Password ay isang secure na 16-character code na binubuo ng Google o Yahoo. Ipinapahintulot nito sa AutoInbox na makabasa at makasagot ng email nang ligtas.

🔒 Bakit Ligtas Ito?
  • Hindi mo ibinibigay sa amin ang iyong totoong password ng email.
  • Kailanman ay hindi namin mababago ang settings ng iyong email account.
  • Maaari mo itong burahin o i-revoke anumang oras sa iyong Google/Yahoo Security page kung ayaw mo nang gamitin ang AutoInbox.

🔴 Gmail Setup Guide

Para sa mga gumagamit ng @gmail.com

  1. Siguraduhing naka-on ang 2-Step Verification sa iyong Google Account.
  2. Pumunta sa pahina ng App Passwords ng Google: Buksan ang App Passwords →
  3. Mag-log in kung hingan ka ng password.
  4. Mag-enter ng pangalan para sa app (i-type ang: AutoInbox) at i-click ang Create.
  5. Lalabas ang isang yellow box na may 16-character code (halimbawa: abcd efgh ijkl mnop).
  6. Kopyahin ito at i-paste sa AutoInbox connect box (tanggalin ang mga patlang o spaces).

🟣 Yahoo Mail Setup Guide

Para sa mga gumagamit ng @yahoo.com

  1. Mag-log in sa iyong Yahoo account at pumunta sa Account Security settings page: Buksan ang Yahoo Security →
  2. I-scroll pababa at i-click ang Generate app password (o Manage app passwords).
  3. I-type ang AutoInbox bilang pangalan ng app at i-click ang Generate.
  4. I-copy ang lalabas na 16-character code.
  5. I-paste ang code sa AutoInbox connect box (tanggalin ang mga patlang o spaces).

💡 Karaniwang Problema at Solusyon (Troubleshooting)

❌ Error: "Invalid credentials" o hindi tinatanggap ang password?

Siguraduhing kopyahin ang buong 16 na letra nang walang spaces. Huwag gamitin ang inyong totoong password sa email, tanging ang App Password lamang ang gumagana.

❌ Walang "App Passwords" option sa Google?

Hindi lalabas ang App Passwords option kung hindi pa naka-on ang 2-Step Verification sa inyong Google Account. I-on muna ito sa Google Security Settings bago bumalik.

🔌 Paano ko ito ididiskonekta?

Maaari mong idiskonekta ang iyong email anumang oras sa pamamagitan ng pag-click sa Disconnect button sa settings ng AutoInbox, o kaya ay burahin ang nabuong App Password sa inyong Google/Yahoo Security page.
